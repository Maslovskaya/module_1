# module_1

Логика выполнения:

Импорт библиотек: Используем pandas для работы с данными и numpy для дополнительных операций.
Загрузка и очистка данных: Загружаем CSV-файл, преобразуем дату в формат datetime.
Ответы на вопросы:
Для вопросов 1-12 используем метод idxmax() и idxmin() для нахождения индекса строки с максимальным/минимальным значением.
Для вопроса 6 проверяем наличие ничьих по столбцу home_team_outcome.
Для вопроса 13 объединяем домашние и гостевые команды и считаем количество игр для каждой.
Для вопросов 14-17 фильтруем данные по регулярному сезону, считаем победы и проигрыши для домашних и гостевых команд, затем объединяем результаты.
Для вопроса 18 используем метод cov() для расчета ковариации между количеством зрителей и победой домашней команды.
Для вопроса 19 считаем количество проигрышей домашних команд по дням недели и вычисляем долю суббот и воскресений.
Для вопроса 20 фильтруем игры с температурой ниже 0°F и сравниваем среднее количество ранов в этих играх со средним по всем играм.
